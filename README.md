# Owlbear Painting Tools

An [Owlbear Rodeo](https://www.owlbear.rodeo/) extension that adds some more ways to draw shapes.

## Installation

1. Go to [Owlbear Rodeo](https://www.owlbear.rodeo/) and click "Add Extension" at the bottom of your profile
2. Add the url `https://painting-tools.owlbear.davidsev.co.uk/manifest.json`
3. In your room click the `...` menu at the bottom left, and select "Extensions" to enable it

## Usage

See [the store page](https://extensions.owlbear.rodeo/painting-tools) for more information.
